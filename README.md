The Digital Ambassador project covers five modules that build essential digital skills for students. It begins with digital literacy awareness by creating an infographic using tools like Canva. It then focuses on developing a professional online presence through platforms such as GitHub and LinkedIn. The project also introduces coding and collaboration using HackerRank and Google Forms. Additionally, it emphasizes professional email communication and proper online etiquette. Finally, it raises awareness about cybercrime and promotes safe digital practices, helping students become responsible and skilled digital users.
